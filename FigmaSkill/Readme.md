# Figma Skill
## Figme Shortcut
- F - activates the Frame tool
- Shift + command + K - open a file
- Command + d - duplicate
- Shift + A - making a frame (Auto Layout)
- Option + command + g - making a general frame
- Command  + \ - hide tool
- Shift + g - hide grid
